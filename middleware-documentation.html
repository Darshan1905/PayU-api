PayU Middleware API

Base URL: https://connect.pavokart.com  ·  Version 1.2.2

Headers (all API requests)

Accept: application/json
X-API-Key: your-api-key

For POST requests, also add: Content-Type: application/json

Auto customer details (create-order)

If customer is omitted or incomplete, Indian name, mobile, email, and address are auto-generated. Shipping mirrors billing. Response includes customer and customerAutoGenerated: true.

1. List products

GET/api/payu/products?page=1&per_page=50

Full URL
https://connect.pavokart.com/api/payu/products?page=1&per_page=50
Headers
Accept: application/json
X-API-Key: your-api-key

2. Create order — simple product

POST/api/payu/create-order

Full URL
https://connect.pavokart.com/api/payu/create-order
Headers
Content-Type: application/json
Accept: application/json
X-API-Key: your-api-key
Body
{
  "lineItems": [
    { "productId": 123, "quantity": 1 }
  ]
}

Redirect user to checkoutUrl in response.

3. Create order — variable product

POST/api/payu/create-order

Body
{
  "lineItems": [
    {
      "productId": 20600,
      "variationId": 20601,
      "quantity": 1
    }
  ]
}

4. Create order — mixed cart

POST/api/payu/create-order

Body
{
  "lineItems": [
    { "productId": 123, "quantity": 1 },
    { "productId": 20600, "variationId": 20601, "quantity": 2 }
  ]
}
